# サンプル
- `font.tex`：フォントの埋め込み設定の確認
- `exam_p.tex`：校内模試の問題．必ず最新版を取得してから問題作ること！
- `exam_s.tex`：校内模試の解答．必ず最新版を取得してから解答作ること！
- `figure`：バネとか．
- `circuitikz`：回路図描く．

# 取得方法
最新版のダウンロード方法（初回）は，
```sh
git clone https://github.com/tetsu-osaka-physics/samples.git ~/Desktop/samples
```
次回以降は
```sh
cd ~/Desktop/samples && git pull
```
（エラーが出る場合は`samples`フォルダを削除してから初回ダウンロード）
